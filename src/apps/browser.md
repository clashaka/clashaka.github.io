# 网页浏览

## Chrome

是一款快捷、安全、定制的浏览器。

[前往下载](https://www.google.com/chrome/)

## Edge

是一款 AI 驱动的浏览器，提供更智能的保护、创建、查找、浏览、游戏、学习和奖励体验。

[前往下载](https://www.microsoft.com/zh-cn/edge)

## Firefox

是一款专注于保护重要数据的浏览器，没有晦涩难懂的隐私保护政策，更不向广告商预留后门。

[前往下载](https://www.mozilla.org/zh-CN/firefox/new/)

## Opera

Opera 网页浏览器提供了更快、更安全、更智能的浏览体验，并提供了免费的 VPN、广告屏蔽和浏览器 AI。

[前往下载](https://www.opera.com/zh-cn)
