# 扩展工具

## Buster

Buster 是一款 Chrome 浏览器的开源工具，利用语音识别自动完成 reCAPTCHA 验证，帮助用户节省时间。

[前往下载](https://github.com/dessant/buster)

## Adblock Plus

是一款广泛使用的广告拦截浏览器扩展程序，可以在各种主流浏览器上使用，如 Chrome、Firefox、Edge 等。其主要功能是拦截网页上的广告，包括横幅广告、视频广告、弹窗广告等。

[前往下载](https://adblockplus.org/)

## TeX All the Things

是一款 Chrome 浏览器扩展程序，可在任何网页上启用 LaTeX 数学公式的渲染。该扩展利用 MathJax 引擎自动识别并渲染网页上的 LaTeX 数学公式，包括内联公式和显示模式公式。

[前往下载](https://chromewebstore.google.com/detail/cbimabofgmfdkicghcadidpemeenbffn)
