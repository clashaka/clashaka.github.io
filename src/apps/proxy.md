# 代理软件

## v2rayN

是一款适用于 Windows 用户的代理软件，它支持多种协议、用户友好界面、订阅功能，并提供高级自定义选项，适合各种水平的用户。

[前往下载](https://github.com/2dust/v2rayN)

## v2rayNG

v2rayNG 是一个为 Android 用户提供的功能强大且易于使用的代理软件客户端，它支持多协议，提供简洁的界面和订阅管理功能，适用于需要在移动设备上进行网络代理的用户。

[前往下载](https://github.com/2dust/v2rayNG)

## Clash Verge Rev

是 Clash Verge 的继任者，一个跨平台的 GUI 代理客户端工具，基于 Tauri 的 Clash Meta 内核开发，支持多种代理协议，具有强大的分流规则功能，并持续更新以确保稳定性和安全性，非常适合需要在不同操作系统上进行网络代理的用户。

[前往下载](https://github.com/clash-verge-rev/clash-verge-rev)

## Clash Meta for Android

是一款基于最新 Clash Meta(mihomo) 内核的 Android 代理客户端软件，具备强大的分流规则和多协议支持，界面友好且支持订阅导入，为需要在手机上进行高效网络代理的用户提供了一个安全、易用的选择。

[前往下载](https://github.com/MetaCubeX/ClashMetaForAndroid)
